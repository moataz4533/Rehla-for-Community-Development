# دليل تحسينات نموذج التبرع — رحلة للتنمية المجتمعية

أضافت هذه النسخة ميزتين ترفعان معدّل التبرع (بناءً على أفضل ممارسات 2026):

## 1. مبالغ مقترحة بسياق

للحالات ذات الهدف المالي، يمكنك الآن إضافة مبالغ جاهزة مع شرح أثر كل
مبلغ. مثال:
- ٢٥٠ جنيه → شهر أدوية لمريض
- ٥٠٠ جنيه → نصف تكلفة جلسة علاج
- ١٠٠٠ جنيه → عملية كاملة

هذه المبالغ تظهر للمتبرع كأزرار سريعة، مع إبقاء خيار إدخال مبلغ حر.
**لماذا؟** الأبحاث تؤكد أن ربط المبلغ بأثر ملموس يرفع التحويل بشكل كبير.

### كيف تضيفها؟
في لوحة التحكم → الحالات → عند إضافة/تعديل حالة بهدف مالي، ستجد قسم
"مبالغ مقترحة بسياق". اضغط "إضافة مبلغ"، أدخل القيمة والسياق.

## 2. التبرع الشهري المتكرر

يمكنك تفعيل خيار "التبرع الشهري" لأي حالة. عندها يرى المتبرع زرّين:
"مرة واحدة" و"شهري متكرر". التبرع المتكرر مصدر دخل مستقر للمؤسسة.

### كيف تفعّلها؟
في نموذج الحالة، فعّل خيار "السماح بالتبرع الشهري المتكرر".

> ملاحظة: التنفيذ الفعلي للتجديد الشهري يتم عبر بوابة الدفع عند ربطها.
> حاليًا النظام يسجّل نوع التبرع (مرة/شهري) ويجهّز كل شيء للربط.
